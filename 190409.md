2. 특수문자 / 문자 / 숫자 포함 형태의 8~15자리 이내의 암호 정규식

   `const regex = /^.*(?=^.{8,15}$)(?=.*\d)(?=.*[a-zA-Z])(?=.*[!@#$%^&+=]).*$/;`

    

   숫자와 문자 포함 형태의 6~12자리 이내의 암호 정규식

   `const regex = /^[A-Za-z0-9]{6,12}$/;`

   

   이메일 정규식
   `var regExp = /^[0-9a-zA-Z]([-_.]?[0-9a-zA-Z])*@[0-9a-zA-Z]([-_.]?[0-9a-zA-Z])*.[a-zA-Z]{2,3}$/i;`
